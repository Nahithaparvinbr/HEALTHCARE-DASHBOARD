# HEALTHCARE-DASHBOARD

This Healthcare Dashboard is an interactive Power BI project created to analyze hospital data and monitor overall healthcare performance. The dashboard provides a clear overview of patient admissions, treatment costs, length of stay, department performance, and doctor distribution. It helps in understanding patient trends, financial performance, and operational efficiency in a hospital setting. The data was cleaned and structured in Excel before building the dashboard, and DAX measures were used to calculate key performance indicators.

Key Highlights

Total Patients: 1000
Average Treatment Cost: ₹38.10K
Average Length of Stay: 7.96 days
Total Doctors: 8
Department-wise patient distribution
Gender-based patient analysis
Admission trends by department
Comparison of treatment cost and stay duration
Interactive filters for Patient ID, Department, Gender, Doctor Name, Admission Date, and Discharge Date
